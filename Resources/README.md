# Boot ROM

Place the NorthStar Advantage Boot ROM here as `AdvantageBootRom.bin`.

This file is not included in the repository for copyright reasons. It can be found in various vintage computing archives. See [djones60/NorthStar-Advantage](https://github.com/djones60/NorthStar-Advantage) for ROMs and documentation.

Alternatively, place it at `~/Library/Application Support/NorthMac/AdvantageBootRom.bin`.
