# Disk Images

Place NorthStar Advantage floppy disk images (`.NSI`) in the subdirectories here:

- `Bootable/` — CP/M boot disks (e.g., advf2, CPM Master, WORDSTAR)
- `Non-Bootable/` — Data and utility disks

These files are not included for copyright reasons. A good collection of NorthStar Advantage disk images is available from [Dave Dunfield's classiccmp archive](http://dunfield.classiccmp.org/img42841/d/nsadvan.zip).

Alternatively, place them at `~/Library/Application Support/NorthMac/Disk Images/`.

**Note:** NorthStar *Horizon* disk images are for different hardware and will not boot.
